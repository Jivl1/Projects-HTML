## Assignment

Homework 6 – HTML & CSS Introduction  
Building a landing page using HTML and CSS to demonstrate:

- Proper HTML document structure
- All essential HTML tags (headings, paragraphs, images, links, lists, tables, forms)
- External, internal, and inline CSS
- CSS selectors (class, ID, element)
- Div and section layout structure

## Project Structure

```
/
├── index.html    ← Main HTML page
├── styles.css    ← External stylesheet
└── README.md     ← This file
```

## How to Run

Open `index.html` directly in any modern web browser. No build tools required.

## Features Demonstrated

| Requirement | Implementation |
|---|---|
| HTML structure | `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` |
| Headings | h1 through h6 used throughout |
| Paragraph | `<p>` tags in every section |
| Image | Hero image with `alt` attribute |
| Links | Nav links, external NASA link, footer links |
| Unordered list | Mission feature lists with `<ul>` / `<li>` |
| Ordered list | Steps list with `<ol>` / `<li>` |
| Table | Fleet overview with `<table>`, `<thead>`, `<tbody>` |
| Form | Contact form with text, email, select, radio, checkbox |
| Navigation | Fixed `<nav>` with anchor links |
| External CSS | `styles.css` linked via `<link>` |
| Internal CSS | `<style>` block in `<head>` |
| Inline CSS | `style=""` attributes on select elements |
| Class selectors | `.btn-primary`, `.mission-card`, `.badge`, etc. |
| ID selectors | `#hero`, `#card-alpha`, `#fleet`, etc. |
| Element selectors | `body`, `h1`, `nav`, `table`, `a`, `p`, etc. |
| Div/section layout | Grid layouts, flexbox, two-column sections |
